image_utils module¶

This image_utils() module is one of the main components related to image processing in our application. This module has various utilities like rescaling ROI, to annotate an image, to check whether an object is intersecting with the selected area or not, etc.

image_utils.annotate_image(image, file_details, selected_area, area_threshold)¶
This function is mainly used to draw boxes and label images with its respective classes that have been identified in the image. It also draws the area selected by the user on the frame.
Inputs:
image

The current frame

type image

image

file_details

All details related to the current frame

type file_details

dict

selected_area

Co-ordinates of the area selected by the user

type selected_area

int

area_threshold

Threshold of the area intersecting

type area_threshold

int

Returns:

Boolean (true or false)
