mongo module¶

This mongo() module contains necessary and supporting function used to manipulate data from and into our mongoDB database. It has provisions to add new data, delete past data, as well as to update the information in a .csv file.

mongo.add_document(database, camera_id, frame, date_time, anomaly_status, classes, boxes, confidence)¶

Storing anomaly data. This data is later used to load anomalies and load objects.

Inputs:
database

The database name

camera_id

The Camera name

type camera_id

str

frame

The current frame

type frame

image

date_time

The current date_time

type date_time

date_time

anomaly_status

Anomaly status whether anomaly is present or not

type anomaly_status

bool

classes

The classes of objects detected

type classes

str

boxes

Coordinates of the bounding boxes of the objects detected

type boxes

list

confidence

The confidence scores for the objects detected by the model

type confidence

int

Returns:

Stores all the information in MongoDB database

mongo.delete_past_data(past_date)¶

This function is used to delete an entire day’s data from MongoDB database for a specific date

Inputs:

Takes the date and time parameter searches in the db :past_date: The date for which we want to delete the data :type past_date: datetime

Returns:

Deletes the data and returns the number of documents deleted in the process
